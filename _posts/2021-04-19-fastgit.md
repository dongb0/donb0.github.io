今天忍受了 git clone 高达 50kb/s 的下载速度长达十分钟之后，突然想起来以前找到过一个 git 的[国内镜像](1)，只要把 clone 的域名 github.com 部分替换成 hub.fastgit.org 即可。想看更详细的说明或者其他修改配置的方法可以查阅[文档](2)。

用完之后已经 1000% 了（笑
